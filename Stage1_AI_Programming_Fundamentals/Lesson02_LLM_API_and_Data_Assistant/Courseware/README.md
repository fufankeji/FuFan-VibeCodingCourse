- Inline Mode: Real-time code completion

- MCP Server installation and configuration

### 3. Building Visual AI-Driven Task Management with MCP
- MCP and AI assistant integration
- Task management system architecture design

### 4. Cursor Rules Mode Application Techniques
- How Rules mode works
- Global rules vs project rules
- Rule priority and inheritance mechanism

- Analyzing project requirements and code style
