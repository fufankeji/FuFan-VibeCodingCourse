<div align="center">

| 阶段 | 重点 | 关键活动 |
|:---:|:---:|:---:|
| **Phase 1：定义** | 需求 | 市场/竞品调研 → 核心灵感 → 需求梳理 → Figma 原型 |
| **Phase 2：研发** | AI 辅助编码 | AI 前端生成 → DB 架构设计 → API 定义 → Code Review |
| **Phase 3：交付** | 部署 | 接口测试 → CI 流水线 → 容器化 → 生产上线 |
