## ADDED Requirements

### Requirement: 关于我区域布局
关于我 Section 必须（SHALL）位于项目展示区下方，采用左右分栏布局（照片 + 简介），移动端改为上下堆叠。

#### Scenario: 桌面端左右分栏
- **GIVEN** 用户使用桌面浏览器
- **WHEN** 查看"关于我"区域
- **THEN** 左侧展示个人照片，右侧展示个人简介文字，两栏水平排列

#### Scenario: 移动端上下堆叠
- **GIVEN** 用户使用移动端设备（视口宽度 < md 断点）
- **WHEN** 查看"关于我"区域
- **THEN** 照片在上方居中，简介文字在下方，垂直堆叠

#### Scenario: 暗色模式适配
- **GIVEN** 用户切换到暗色模式
- **WHEN** 查看"关于我"区域
- **THEN** Section 背景、文字颜色均切换为暗色系配色

#### Scenario: Section 锚点（边界）
- **GIVEN** 页面加载完成
- **WHEN** 用户通过 `#about` 锚点导航到此区域
- **THEN** 页面平滑滚动到"关于我"区域，内容不被导航栏遮挡

### Requirement: 个人照片展示
关于我区域左侧必须（SHALL）展示一张个人照片，使用 lazy loading。

#### Scenario: 照片正常展示
- **GIVEN** 照片路径存在
- **WHEN** 用户滚动到"关于我"区域
- **THEN** 照片以固定比例（正方形）展示，圆角裁切，延迟加载

#### Scenario: 照片缺失时的降级（边界）
- **GIVEN** 照片路径不存在或加载失败
- **WHEN** 页面渲染"关于我"区域
- **THEN** 照片区域展示占位背景色，不出现破碎图片图标

#### Scenario: 照片加载前无布局偏移（边界）
- **GIVEN** 照片尚未加载
- **WHEN** 用户滚动到"关于我"区域
- **THEN** 照片区域保持固定高宽比占位，加载后无布局跳动

### Requirement: 个人简介文字
关于我区域右侧必须（SHALL）展示 3 段个人简介文字。

#### Scenario: 简介完整展示
- **GIVEN** 页面加载完成
- **WHEN** 用户查看简介区域
- **THEN** 展示 3 段文字，段落之间有适当间距，文字可读性良好

#### Scenario: 长文本不溢出（边界）
- **GIVEN** 某段文字较长
- **WHEN** 在不同屏幕宽度下查看
- **THEN** 文字自动换行，不产生水平溢出

### Requirement: 品牌标签展示
关于我区域下方必须（SHALL）展示品牌标签「赋范空间」。

#### Scenario: 品牌标签正常展示
- **GIVEN** 页面加载完成
- **WHEN** 用户查看"关于我"区域底部
- **THEN** 品牌标签「赋范空间」居中展示，字号突出，视觉上作为品牌锚点

#### Scenario: 暗色模式下品牌标签（边界）
- **GIVEN** 用户切换到暗色模式
- **WHEN** 查看品牌标签
- **THEN** 标签颜色切换为暗色系配色，保持可读性和视觉突出
